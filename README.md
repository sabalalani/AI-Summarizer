## üåü Features

- **Multiple Model Support**: Choose between different transformer models for summarization
- **Customizable Length**: Adjust minimum and maximum summary length with sliders
- **Real-time Processing**: Get instant summaries with progress indicators
- **Web Interface**: Clean, responsive Bootstrap-based UI
- **FastAPI Backend**: High-performance asynchronous API
- **Hugging Face Integration**: Deployable on Hugging Face Spaces
